# ハードウェアアーキテクチャ資料：水槽管理システム

## 📋 概要

本システムは**多層防御**トポロジーを採用し、入力側（AC）でのノイズフィルタリング、メインバス（DC）での安定化、エッジ側（アクチュエーター）での誘導性スパイク抑制を行います。

```
┌─────────────────────┐     ┌─────────────────────┐     ┌─────────────────────┐
│ レイヤー1: AC        │ ──► │ レイヤー2: DC        │ ──► │ レイヤー3: エッジ    │
│ 入力/保護           │     │ バス/ロジック       │     │ 周辺機器/抑制       │
│                     │     │                     │     │                     │
│ • ACヒューズ        │     │ • 12.53V電源        │     │ • フライバックD     │
│ • NTC（突入電流）   │     │ • LM2596 → 5.1V    │     │ • デカップリングC   │
│ • EMIフィルタ（Y）  │     │ • コンデンサバンク  │     │ • センサー（GX12）  │
│ • キャニスターリレー│     │ • 8ch MOSFET        │     │ • ポンプ + バルブ   │
└─────────────────────┘     └─────────────────────┘     └─────────────────────┘
```

---

## ⚡ レイヤー1：AC入力・保護（インフラストラクチャ）

電力入力アーキテクチャはシンプルさと最大限の電気的安全性を重視し、重いフィルタリング（EMIフィルターやNTC/突入電流制限など）は、スイッチング電源にすでに組み込まれているプロ用回路に委ねています。

### コンポーネント

| コンポーネント | 機能 |
|---|---|
| **ACガラス管ヒューズ**（3A〜5A） | 必須項目：短絡および火災リスクに対する保護 |
| **SSRリレーモジュール 1CH** | キャニスターフィルター独立制御（ESP32経由、AC） |

### 配線図

```text
[ IEC C14 入力 ]
      │
      ├─── [ 接地ピン (緑) ] ─────────────────┬───► [ 電源 接地 (G) 端子 ]
      │                                       │
      │                                       └───► [ キャニスター出力 接地ピン ]
      │         
      ├─── [ 中性ピン (青) ] ─────────────────┬───► [ 電源 中性 (N) 端子 ]
      │                                       │
      │                                       └───► [ キャニスター出力 中性ピン ]
      │
      └─── [ 相ピン (茶) ] ───── [ ヒューズ ] ──┐
                                                │
             ┌──────────────────────────────────┘
             │                                  
             ├───────────────────► [ 電源 相 (L) 端子 ]
             │
             └───► [ SSRリレー ネジ1 ] ── スイッチ ──► [ SSRリレー ネジ2 ] ──► [ キャニスター出力 相ピン ]
```

## 🔋 レイヤー2：DCバス・ロジック（分配）

電力をロジックレベルに変換し、重負荷スイッチング時のESP32安定性を維持します。

### コンポーネント

| コンポーネント | 機能 |
|---|---|
| **スイッチング電源 180W** | 12.53Vに調整 |
| **ヒューズ T5AL250V** | 8台のポンプとセンサーの物理的保護 |
| **LM2596** | 5.1Vに調整されたステップダウン（ESP32電源） |
| **1× 470µF 16V** | MOSFETの12V入力に並列（ポンプ起動時の電圧降下を防止） |
| **4× 1000µF 10V** | 5V出力に並列（ESP32のスパイクや変動対策、「無停電電源」として安定動作を保証） |

### 配線図

```text
[ 電源 12.53V ]
   │              │
 (V+)           (V─) ──────────────────────────────┐ (スターGND)
   │              │                                │
[T5Aヒューズ]     │                                │
   │              │                                │
   ├──────────────┼────────────────┐               │
   │              │                │               │
[LM2596 IN+]  [LM2596 IN─]    [MOSFET VIN]   [MOSFET GND]
   │              │                │               │
(出力 5.1V)       │         (1× 470µF 16V)         │
   │              │                │               │
(4× 1000µF 10V)   │                │               │
   │              │                │               │
[ESP32 VIN]   [ESP32 GND]          │               │
   │              │          [ 8ch MOSFET ]        │
   │              └────────────────┴───────────────┘
```

> [!IMPORTANT]
> **スターGND**：全ての負側基準は電源のV−端子に直接戻し、グランドループを防止してください。

## 🌊 レイヤー3：周辺機器・抑制（エッジ）

誘導性ノイズ（フライバック）の軽減および長距離ケーブル（1.2m）でのセンサー読み取り安定化を担当します。

### コンポーネント

| コンポーネント | 用途 | 数量 |
|---|---|---|
| **FR154ダイオード**（高速リカバリ） | 給排水ポンプ | 2 |
| **FR154ダイオード**（高速リカバリ） | ペリスタルティックポンプ（肥料4 + プライム1） | 5 |
| **1N5822ダイオード**（ショットキー） | ソレノイドバルブ（チャンネル8） | 1 |
| **10µF / 22µF 50Vコンデンサ** | ケーブル端でのセンサーデカップリング（GX12） | — |

> [!NOTE]
> **合計：8個のフライバックダイオード** — モーターチャンネルごとに1個（7× FR154 + 1× 1N5822）。すべてケーブル端のモーター近傍に設置。

### 配線図 — アクチュエーター

```
[ MOSFETチャンネル ] ──────── (1.2mワイヤ) ──────┬────────── (ポンプ +)
                                                 │
                                           [ フライバックD ]
                                           (ストライプは+側)
                                                 │
[ バスGND ] ─────────────── (1.2mワイヤ) ────────┴────────── (ポンプ ─)
```

> [!CAUTION]
> **フライバックダイオード**：必ず**ケーブル端**（モーター近傍）に設置してください。1.2mケーブルがアンテナのようにノイズを放射するのを防ぎます。

### 配線図 — 超音波センサー（JSN-SR04T）

センサーは5Vで動作しますが、ESP32のGPIOは最大**3.3V**です。**ECHO**ピンにはGPIO34に入力する前に5V → 3.3Vに降下させる分圧回路が必要です。

| コンポーネント | 機能 |
|---|---|
| **1kΩ抵抗**（R1） | ECHOとGPIO34の間に直列 |
| **2kΩ抵抗**（R2） | GPIO34とGND間のプルダウン |

**計算式：** `Vout = 5V × R2 / (R1 + R2) = 5 × 2k / 3k = 3.33V` ✅

```
                           ┌──────────────────┐
  ESP32 GPIO18 (TRIG) ─────┤ TRIG    JSN-SR04T│
                           │                  │
  ESP32 5V ────────────────┤ VCC              │
                           │                  │
  GND ─────────────────────┤ GND              │
                           │                  │
                           │ ECHO ────┐       │
                           └──────────┼───────┘
                                      │
                                 [ R1 = 1kΩ ]
                                      │
                    ESP32 GPIO34 ──────┤
                                      │
                                 [ R2 = 2kΩ ]
                                      │
                                    (GND)
```

> [!WARNING]
> **ECHOを分圧回路なしでESP32に直接接続しないでください。**5VのECHO信号はESP32のGPIOを永久に損傷する可能性があります。

### 配線図 — 静電容量式水位センサー（XKC-Y25-NPN）

ガラス/チューブ壁越しに液体の存在を検知する非接触センサー。NPN オープンコレクタ出力、ESP32の3.3Vと直接互換性があります。

| センサーピン | 接続先 |
|---|---|
| **VCC**（赤） | ESP32 3.3V |
| **GND**（黒） | GND |
| **OUT**（黄） | GPIO 4（`INPUT_PULLUP`） |

```
  ESP32 3.3V ─────────── VCC（赤）
                                            ┌────────────────┐
  ESP32 GPIO4 ─────────── OUT（黄） ───────┤ XKC-Y25-NPN    │
   (INPUT_PULLUP)                          │ （ガラス外側   │
                                            │  に貼付）      │
  GND ─────────────────── GND（黒） ───────┤                │
                                            └────────────────┘
```

> [!TIP]
> **外部抵抗は不要。**ESP32の内部プルアップ（約45kΩ）でNPNオープンコレクタに十分です。センサーは水槽ガラスの**外側**、希望する最大水位に貼り付けます。

### 配線図 — TFT ST7735ディスプレイ (SPI)

1.8インチカラーディスプレイ、128×160ピクセル。**3.3V**で動作 — 5Vは非互換。カスタムピンでソフトウェアSPI（ビットバンギング）を使用します。

| ディスプレイのピン | ESP32接続 | GPIO | 備考 |
|---|---|---|---|
| **VCC** | 3.3V | — | ⚠️ 絶対に5Vを使用しない |
| **GND** | GND | — | |
| **CS** | D15 | GPIO15 | チップセレクト (Chip Select) |
| **RESET** | ENピン | — | ESP32と共有のハードウェアリセット |
| **A0 (DC)** | TX2 | GPIO17 | データ/コマンド (Data/Command) |
| **SDA (MOSI)** | D23 | GPIO23 | SPIデータ |
| **SCK** | RX2 | GPIO16 | SPIクロック |
| **LED** | 3.3V | — | 固定バックライト (空きGPIOなし) |

```
ESP32 3.3V  ────►  VCC
ESP32 GND   ────►  GND
ESP32 D15   ────►  CS
ESP32 EN    ────►  RESET
ESP32 TX2   ────►  A0 (DC)
ESP32 D23   ────►  SDA (MOSI)
ESP32 RX2   ────►  SCK
ESP32 3.3V  ────►  LED (固定バックライト)
```

> [!WARNING]
> ディスプレイの**RESET**ピンはESP32の**EN**ピンに接続してください（GPIOではありません）。これにより、ディスプレイがマイクロコントローラと一緒にリセットされます。ENがHIGH（通常動作）のとき、ディスプレイは正常に動作します。

> [!NOTE]
> ESP32 DevKit V1ブレイクアウトボードでは、**D16**および**D17**はそれぞれ**RX2**および**TX2**とラベル付けされています。端子台のラベルを使用して正しい位置を確認してください。

### 配線図 — ナビゲーションボタン

ディスプレイを制御する物理ボタン：短押しでページ切り替え。長押しでメニューを開く（TPA開始またはメンテナンスモード）。

| ピン | 接続 | GPIO | 設定 |
|---|---|---|---|
| **端子 1** | GPIO19 | GPIO19 | `INPUT_PULLUP` — LOWでアクティブ |
| **端子 2** | GND | — | |

```
ESP32 GPIO19 ─────── [ ボタン ] ──── GND
         INPUT_PULLUP, LOWでアクティブ (押下 = LOW)
```

> [!TIP]
> ボタンは標準的なプッシュボタンまたはタクタイルスイッチを使用してパネルに取り付けることができます。ファームウェアは**短押し**（ページ変更）と**長押し > 1秒**（メニューを開く）を検出します。

## 🛠️ 安全な実装上の注意点

1. **AC 接続** — 110V/220Vのすべてのハンダ付け接点および接続部は、最大限の安全性を確保するために熱収縮チューブで絶縁してください。

2. **コンデンサの極性** — すべての電解コンデンサの負極ストライプを確認してください（特に5.1Vで動作する1000µF 10V）。

3. **フライバックダイオード** — 必ずケーブル端（モーター近傍）に設置し、1.2mケーブルがアンテナのようにノイズを放射するのを防いでください。

4. **スターGND** — すべての負側基準は電源のV−端子に直接戻し、グランドループを防止してください。

5. **サイフォン現象（水力学）** — 排水ポンプ停止後に重力で水槽が排水され続けるのを防ぐため、排水ポンプと並列にソレノイドバルブを配線する（同じMOSFETチャンネルに接続し、それぞれにフライバックダイオードを取り付ける）か、ホースに空気穴を開けてください。
